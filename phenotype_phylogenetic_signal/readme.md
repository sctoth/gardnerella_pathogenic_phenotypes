# Phenotype Phylogenetic Signal
**Purpose:** To assess the phylogenetic association of measured *Gardnerella* phenotypes related to pathogenicity.

**Overview:**  

Phylogenetic signal – the tendency of closely related species to exhibit similar trait values (Münkemüller et al., 2012) – was assessed for observed phenotypes in *Gardnerella spp.* strains using Moran’s I (Moran 1950) and Pagel’s λ (Pagel 1999). 
For each observed phenotype across 22 *Gardnerella* spp. strains, phylogenetic signal analyses were conducted using **phylosignal** (v1.3.1). Moran's I and Pagel's Lambda tests were performed to evaluate whether phenotypic traits are non-randomly distributed across the phylogeny.

* `phylogenetic_signal_testing.Rmd`: R code to test phylogenetic signal in measured phenotypes
* `phenotype_distributions.Rmd`: R code to plot replicate values and average across isolates to assess phenotype distribution per strain
